# Цель:

### Написать приложение на flutter с урезанным функционалом сервиса "КиноПоиск", где нужно реализовать:

- сохранение и отображение списка любимых фильмов;
- использование SQLite для хранения данных о фильмах;
- добавление и удаление фильмов из списка;
- сохранение дополнительных данных о каждом фильме (например, год выпуска и жанр и т.п.);
- редактирование информации о фильме;
- добавление картинки к фильмам и их отображение;
- в настройках добавить переключатели для выбора темы приложения (светлая/темная);
- сохранять выбранную тему в Shared Preferences и применять при запуске приложения.
